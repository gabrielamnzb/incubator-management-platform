# 📖 About the Project

The **Incubator Management Platform** is a Software Engineering case study developed for the **PampaTec Technology Incubator**.

The project aims to digitalize and optimize the startup selection process by replacing manual procedures and spreadsheet-based workflows with a centralized platform supporting entrepreneurs, incubator managers, evaluators and researchers throughout the incubation process.

Rather than focusing exclusively on software implementation, this repository documents the complete engineering process behind the conception of the platform, demonstrating how a real-world information system was specified, modeled and validated.

# 👥 User Profiles

The platform was designed around four user profiles, each with dedicated workflows and responsibilities.

### 👤 Entrepreneur

- Guided application wizard
- Business plan submission
- Proposal tracking
- Mentoring feedback
- Appeals process

### 📋 Incubator Manager

- Proposal dashboard
- Kanban workflow
- Document validation
- Evaluation scheduling
- Selection management

### 📝 External Evaluator

- Proposal analysis
- Evaluation forms
- Technical scoring
- Decision support

### 🔬 Professor & Researcher

- Research partnership management
