# 📖 Sobre o Projeto

O **Portal de Editais PampaTec (PEP)** é um estudo de caso em Engenharia de Software desenvolvido para a **Incubadora Tecnológica PampaTec**.

O projeto tem como objetivo digitalizar e otimizar todo o processo de seleção de startups, substituindo procedimentos manuais e controles realizados em planilhas por uma plataforma centralizada que atende empreendedores, gestores da incubadora, avaliadores e pesquisadores.

Mais do que um protótipo de software, este repositório documenta todo o processo de engenharia utilizado na concepção da solução, demonstrando como um sistema real foi especificado, modelado, validado e documentado seguindo boas práticas de Engenharia de Software.

# 👥 Perfis de Usuário

A plataforma foi projetada para atender quatro perfis principais de usuários.

### 👤 Empreendedor

- Assistente guiado para inscrição
- Submissão do plano de negócios
- Acompanhamento da proposta
- Visualização de feedbacks das mentorias
- Interposição de recursos

### 📋 Gestor da Incubadora

- Dashboard gerencial
- Fluxo Kanban
- Validação documental
- Agendamento de avaliações
- Gestão do processo seletivo

### 📝 Avaliador Externo

- Interface para análise das propostas
- Formulários padronizados de avaliação
- Processo de pontuação técnica
- Apoio à tomada de decisão

### 🔬 Professores e Pesquisadores

- Acompanhamento das startups
- Gestão de parcerias de pesquisa
- Integração com o ecossistema de inovação
