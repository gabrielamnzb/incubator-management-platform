# Protótipos

Esta pasta contém os protótipos de alta fidelidade desenvolvidos no Figma.

Os protótipos representam as interfaces dos diferentes perfis de usuário do sistema:

- Empreendedor
- Gestor da Incubadora
- Comissão Avaliadora
- Professor/Pesquisador
